# 🏢 Landing Page - Planejar Patrimonial

Landing page moderna e responsiva desenvolvida para a **Planejar Patrimonial LTDA**, empresa especializada em planejamento patrimonial, gestão financeira e gestão de cobrança.

## 📋 Sobre o Projeto

Este projeto apresenta uma landing page profissional com design elegante, utilizando as cores institucionais da empresa (dourado e cinza), com foco em conversão e experiência do usuário.

## ✨ Funcionalidades

- ✅ Design responsivo para todos os dispositivos (mobile-first)
- ✅ Navegação suave entre seções
- ✅ Menu mobile hamburger
- ✅ Animações e efeitos visuais modernos
- ✅ Contador animado de estatísticas
- ✅ Formulário de contato com validação
- ✅ Sistema de notificações
- ✅ Botão "Voltar ao topo"
- ✅ Efeito parallax no hero section
- ✅ Otimização de performance

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização moderna com variáveis CSS, Grid e Flexbox
- **JavaScript** - Interatividade e animações
- **Font Awesome** - Ícones

## 📂 Estrutura do Projeto

```
landing-page_planejar-patrimonial/
│
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # JavaScript
├── logo-gestao.png     # Logo da empresa
└── README.md           # Documentação
```

## 🎨 Seções da Landing Page

1. **Header/Navegação** - Menu fixo com links para as seções
2. **Hero Section** - Apresentação principal com call-to-action
3. **Sobre** - Informações sobre a empresa (missão, visão, valores)
4. **Serviços** - Cards detalhados dos serviços oferecidos:
   - Planejamento Patrimonial
   - Gestão Financeira
   - Gestão de Cobrança
   - Gestão de Imóveis
   - Gestão de Equipamentos
   - Consultoria Empresarial
5. **Diferenciais** - 6 diferenciais competitivos da empresa
6. **Estatísticas** - Números relevantes com animação
7. **Contato** - Formulário e informações de contato
8. **Footer** - Informações da empresa e links rápidos

## 📱 Como Usar

1. Clone ou faça download do repositório
2. Abra o arquivo `index.html` em um navegador moderno
3. Navegue pela landing page

Ou acesse diretamente abrindo o arquivo no navegador:
```bash
# Windows
start index.html

# Linux/Mac
open index.html
```

## 📝 Personalização

Para personalizar a landing page:

1. **Cores**: Edite as variáveis CSS em `styles.css` (seção `:root`)
2. **Conteúdo**: Modifique o texto diretamente em `index.html`
3. **Logo**: Substitua o arquivo `logo-gestao.png`
4. **Formulário**: Configure o envio no `script.js` (função de submit do formulário)
